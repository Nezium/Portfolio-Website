# Portfolio Website

A modern portfolio website built with Next.js, TypeScript, and Tailwind CSS.

## Tech Stack
- Next.js 14
- TypeScript
- Tailwind CSS
- Framer Motion

## Getting Started
```bash
npm install
npm run dev
```
